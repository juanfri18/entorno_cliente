            2. ¿Qué características diferencian a un lenguaje de scripting en entorno cliente de
            un lenguaje en entorno servidor?
                entorno cliente:

                    -Se ejecuta en el navegador del usuario, no en el servidor.
                    -permite manipular el contenido,estilos,eventotos de la maquina...
                    -no necesita conexion con el servidor para ejecutarse.
                    -acceso limitado a recursos del sistema del usuario por razones de seguridad.
                    -El rendimiento y compatibilidad dependen del navegador.
                    - se suele usar mas para la validacion de formularios antes de enviarlos ,aminaciones,efectos,interactividad,
                      comunicacion asincrona con el servidor mediante AJAX/fetch.
                    Un ejemplo tipico es JavaScript.  
                entorno servidor:

                    -Se ejecuta en el servidor web antes de enviar la pagina al cliente.
                    -se encarga de la logica de negocio,gestion de bases de datos,autenticacion,procesamiento de formularios.
                    -necesita conexion con el servidor para ejecutarse.
                    -Acceso a recursos del servidor y bases de datos.
                    -El rendimiento depende de la capacidad del servidor y la carga de trabajo.
                    -se suele usar mas para la gestion de bases de datos,autenticacion de usuarios,procesamiento de formularios,logica de negocio.
                    Un ejemplo tipico es PHP,Python,Node.js.

            3. Explica qué es un framework JavaScript. Pon dos ejemplos y explica brevemente
            para qué se utilizan.
                Un framework JavaScript es una biblioteca de código predefinido que proporciona una estructura y herramientas para 
                desarrollar aplicaciones web de manera más eficiente. Facilitan la creación de interfaces de usuario, la gestión del estado
                de la aplicación y la manipulación del DOM, entre otras cosas.
                Ejemplos:
                    -React.js: Es una biblioteca desarrollada por Facebook que se utiliza para construir interfaces de usuario interactivas y reutilizables.
                               Permite crear componentes que pueden gestionar su propio estado y renderizarse de manera eficiente cuando los datos cambian.
                    -Angular: Es un framework desarrollado por Google que se utiliza para construir aplicaciones web de una sola página (SPA). 
                               Proporciona una estructura completa para el desarrollo, incluyendo enrutamiento, gestión del estado, y herramientas para pruebas y construcción.
            

            
            4. ¿Qué ventajas e inconvenientes tiene usar JavaScript en comparación con otros
            lenguajes de scripting en cliente?
                Ventajas:
                    -Amplia compatibilidad: JavaScript es compatible con todos los navegadores web modernos, lo que lo convierte en una opción universal para el desarrollo web.
                    -Interactividad: Permite crear interfaces de usuario dinámicas e interactivas, mejorando la experiencia del usuario.
                    -Ecosistema rico: Cuenta con una gran cantidad de bibliotecas y frameworks (como React, Angular, Vue) que facilitan el desarrollo.
                    -Ejecución en el cliente: Al ejecutarse en el navegador del usuario, reduce la carga en el servidor y mejora la velocidad de respuesta.
                Inconvenientes:
                    -Seguridad: Al ser un lenguaje que se ejecuta en el cliente, puede ser vulnerable a ataques como XSS (Cross-Site Scripting).
                    -Compatibilidad entre navegadores: Aunque la mayoría de los navegadores soportan JavaScript, existen diferencias en la implementación que pueden 
                         causar problemas.
                    -Rendimiento: En aplicaciones muy complejas, el rendimiento puede verse afectado si no se optimiza adecuadamente.
                    -Dependencia del usuario: Si un usuario desactiva JavaScript en su navegador, ciertas funcionalidades de la página web pueden no funcionar correctamente.

            5. Describe tres usos habituales de JavaScript en una página web moderna.
                -Validación de formularios: JavaScript se utiliza para validar los datos ingresados por los usuarios en formularios antes de enviarlos al servidor, 
                    asegurando que la información sea correcta y completa.
                -Interactividad y animaciones: Permite crear efectos visuales, animaciones y respuestas interactivas a las acciones del usuario,
                    mejorando la experiencia de navegación.
                -Comunicación asíncrona con el servidor: A través de tecnologías como AJAX o Fetch API, JavaScript puede enviar y
                    recibir datos del servidor sin necesidad de recargar la página, permitiendo una experiencia más fluida y dinámica.
